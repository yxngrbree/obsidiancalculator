# ObsidianCalculator

A clean and minimal calculator built with Python.

ObsidianCalculator is a lightweight project designed to perform basic arithmetic operations in a simple and efficient way. It serves as a beginner-friendly Python project while remaining easily extendable for more advanced functionality.

---

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Simple and readable Python code
- Easy to modify and expand

---

## Requirements

- Python 3.x

Check your Python version:

```bash
python --version
```
